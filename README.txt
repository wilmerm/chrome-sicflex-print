Extención para Google Chrome.

Esta extención fue disesñada a medida.

Esta extención permite re-imprimir las facturas en el sistema Sicflex,
mediante el cuadro de impresión del navegador, para así poder selecionar
un formato de factura diferente o exportarla en PDF, etc.

Esta extención solo funciona en: ...sicflex.com, especifícamente
en el módulo "Facturación a Clientes" del sistema Sicflex.
